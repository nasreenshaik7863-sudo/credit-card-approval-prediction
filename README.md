# Credit Card Approval Prediction

## 📌 Project Description
This project predicts whether a credit card application will be approved or rejected based on income and age using Machine Learning.

## 🧠 Algorithm Used
- Random Forest Classifier

## 💻 Technologies Used
- Python
- Pandas
- Scikit-learn

## 🚀 How it works
User enters income and age → Model predicts approval or rejection.

## 📊 Output
- Approved ✅
- Rejected ❌
